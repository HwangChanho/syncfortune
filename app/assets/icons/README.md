# 메뉴 아이콘 제작 가이드 (홈 6개)

> 한지·먹·주 전통 모던 테마용 커스텀 아이콘. 이모지·컬러풀 금지 — **먹(모노톤) 라인/판화 무드**.

## 파일 (정확히 이 이름·경로)
`app/assets/icons/` 아래:

| 파일 | 메뉴 | 모티프(예시) | 색 |
|---|---|---|---|
| `myeongsik.png` | 내 명식 | 사주 4기둥 / 별자리 / 命 도장 | 먹 `#2B2B2B` |
| `manse.png` | 만세력 | 달력 / 고서·두루마리 | 먹 `#2B2B2B` |
| `taro.png` | 타로 | 카드 한 장 / 카드 부채꼴 | 먹 `#2B2B2B` |
| `today.png` | 오늘의 운세 | 해·달 / 일진 | 먹 `#2B2B2B` |
| `premium.png` | 프리미엄 풀이 | 붓·두루마리 / 펼친 책 | **주색 `#C0392B`** |
| `compat.png` | 궁합 | 두 사람 / 매듭(결) / 합(合) | **주색 `#C0392B`** |

## 스펙
- **크기:** 정사각 **144×144 px** (표시 42dp @3x). 동일 캔버스로 통일.
- **포맷:** PNG, **투명 배경**. (SVG 원하면 `react-native-svg` 도입 — 알려주세요)
- **여백:** 캔버스 가장자리 **12~15% 패딩**(아이콘이 끝에 닿지 않게 — 카드에서 잘림 방지).
- **색:** 무료 4개 = 먹 `#2B2B2B` 모노톤 / 프리미엄 2개(`premium`·`compat`) = 주색 `#C0392B`. 그라데이션·컬러 지양.
- **스타일:** 라인(아웃라인) 또는 전통 **판화·도장·붓** 느낌. 굵기 일관(스트로크 동일). 6개가 한 세트로 보이게.
- **톤:** 한지 배경(`#F4EFE6`/카드 `#FBF8F1`) 위에 올라가므로, 너무 옅으면 안 보임 — 먹은 진하게.

## 적용 방법 (daniel이 PNG 6개 넣은 뒤)
`app/src/app/(app)/index.tsx` 의 `MENU` 각 항목에 `image` 추가:
```ts
{ key: 'myeongsik', ..., icon: '命', image: require('../../../assets/icons/myeongsik.png'), ... }
```
`image`가 있으면 한자 fallback 대신 이미지가 자동 렌더된다(코드 변경은 그 require 한 줄씩).

> 미제작 상태에서는 한자(命·曆·占·日·解·合)가 임시로 보인다 — 정상.
