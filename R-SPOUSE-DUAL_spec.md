# R-SPOUSE-DUAL: 배우자 이원 궁위 해석 룰

**상태**: 초안 (Draft v0.1)
**작성 기준**: Boss 골든 레퍼런스 차트 (辛丑 일주) 케이스 스터디 기반
**연계 모듈**: R-INT-LAYER (간지 상호작용), R45 (세운체인), R46 (궁합 ρ엔진), lifegraph 곡선 시스템

---

## 1. 배경 및 문제의식

기존 배우자 해석은 대부분 배우자궁(일지) 단일 기준으로 이루어짐. 그러나 실제 사례 분석 결과, 다음 두 축이 서로 다른 배우자상을 가리키는 경우가 존재함:

- **배우자성(配偶星)**: 재성/관성이 위치한 궁(주로 월지·년지)이 보여주는 "끌리는 이성상" — 사회적 매력, 첫인상, 도화성 관련
- **배우자궁(配偶宮)**: 일지 그 자체 — 결혼생활의 실질적 본질, 항상 최우선 가중치를 갖는 정궁(正宮)

두 축이 일치하지 않고 합/충/파/원진/격각 등의 관계로 얽힐 때, "이상형과 실배우자의 괴리"라는 해석 축이 발생함.

### 1.1 케이스 스터디: Boss 차트

- 년주 甲戌 / 월주 丁卯 / 일주 辛丑 / 시주 丁酉
- 월지 卯(편재) — 년지 戌과 卯戌육합 → 사회적 배경과 결합된 "화려한 이상형" 신호
- 일지 丑(편인) — 시지 酉와 酉丑반합 → 자기 자신과 유사 오행으로 흡수되는 "안정적으로 품어주는" 배우자상
- 卯-丑 관계: **격각** (寅을 사이에 두고 직접 충돌하지 않으나 서로 연결되지 않는 구조)
- 원국에 식상 부재 → 편인의 부작용(과잉 통제·탈식) 발현 대상이 없어 순수하게 안정 지향으로 작동

## 2. 핵심 판단 트리

```
IF 배우자성 위치 == 배우자궁 위치 (또는 직접 합)
  → 이상형 = 실배우자 일치형
     (컨텐츠 프레임: "끌림이 곧 결실")

ELIF 배우자성-배우자궁 관계 == 격각
  → 트랙 분리형: 연애 초반 끌림 ≠ 결혼 상대 속성
     세운 개입 시 시나리오 A(결실) / B(이별)로 분기 (3장 참조)

ELIF 배우자성-배우자궁 관계 == 충
  → 정면충돌형: 관계 내내 긴장 지속, 극단적 선택을 요구하는 시기 존재

ELIF 배우자성-배우자궁 관계 == 원진 / 파
  → 은근한 갈등형: 표면적으로는 무난하나 장기적 피로 누적
```

궁위 원칙: 재성/관성 자체는 배우자 그 자체가 아니라 배우자상이 형성되는 사회적 접점. 실제 결혼 생활의 파트너 판단은 항상 일지(배우자궁)를 우선.

## 3. 세운 개입 판단 표

대운 내에서 세운이 배우자성/배우자궁에 개입할 때의 해석 규칙.

| 세운 조건 | 해석 | 라벨 |
|---|---|---|
| 세운이 배우자궁을 충 | 관계의 실질적 변곡점 (결혼/이별/동거 등 이벤트 발생 가능성 高) | `EVENT_CANDIDATE` |
| 세운이 배우자성과 합(반합 포함) | 해당 유형 인연 활성화·강화 | `TYPE_A_ACTIVE` |
| 세운이 배우자성을 충 | 그 유형 인연 정리·이탈 | `TYPE_A_RESOLVE` |
| 세운이 배우자궁과 합(반합 포함) | 안정형 인연으로 무게중심 이동 / 정착 신호 | `TYPE_B_SETTLE` |
| 세운이 배우자궁과 동일 지지(복음) | 기존 관계 확정 또는 안정형 신규 인연 진입 | `CONFIRM` |

### 3.1 시나리오 분기 원칙

같은 격각 구조 안에서도 실제 만나는 상대가 배우자성형(화려·재성 우세)인지 배우자궁형(안정·인성 우세)인지에 따라 이후 흐름이 갈림. 사주 구조 자체는 방향성만 제공하며, 상대 유형 확정은 R46 궁합 엔진이 담당.

- **시나리오 A (결실)**: 배우자성 활성 세운에 만난 인연이 이후 배우자궁 합 세운을 거치며 정착
- **시나리오 B (이별 후 재정착)**: 배우자성 활성 세운에 만난 인연이 이후 배우자성 충 세운에 정리되고, 다음 대운/세운 주기에서 새로운 배우자궁형 인연으로 재정착

### 3.2 케이스 스터디 적용 결과 (庚午대운, 27~36세)

## 4. 나이대(연상/연하) 보조 해석 — 경향성 스코어

**주의: 신뢰도 낮음 레이어. 확정 로직 아님. 실제 상대 사주 존재 시 R46이 항상 override.**

| 조건 | 경향성 |

| 배우자성이 년지와 결합(합) | 연상 경향 가중치 ↑ |

## 7. 확정 규칙 (daniel 2026-07-20 — §7 미결정 해소)

### 7.1 발동강도지수 / 안착강도지수 (구 '괴리지수' 폐기·재정의)
"충이 제일 강하다" 틀은 유지하되 **두 축으로 분리**한다. 한 관계 = `발동강도 × 안착강도` 두 숫자.

- **발동강도지수**(사건을 터뜨리는 힘 — 만남이든 이별이든):
  - 충 **90**(가장 강함) · 원진 **70**(심리적 반감·발동 약하나 누적 피로) · 파 **50**(구조적 손상·감정 발동 약함) · 격각 **40**(접촉 없는 분리·발동 최저·트랙이 안 만남)
- **안착강도지수**(합/반합 별도 축 — 타입이 정해지는 힘):
  - 육합 **80** · 반합 **50** · 방합 **40**
- 예: 2027 未丑충(발동 90) + 未卯반합(안착 50) → "강하게 시작되고 중간 정도로 타입이 정해지는 해".

### 7.2 시나리오 A/B — 하이브리드 자동판별 (A/B 정의 수정)
- A/B 분기는 이제 "화려/안정 타입 차이"가 **아니라** → **"충으로 발동된 관계가 합으로 안착되는가(A) vs 다음 충에서 재발동=정리되는가(B)"**.
- 기본 = **세운 라벨 기반 확률 자동 제시**(예: "이 시기 발동 후 안착 확률 65%").
- 사용자가 실제 연애 이력(만난 시점·지속 여부) 입력 시 그 실측으로 **override**(R46과 동일 원칙).

### 7.3 나이대 경향성 (합 기반 유지 — 나이대는 발동이 아니라 '관계 성격/포지션'=안착 축)
- 배우자성-년지 육합 **+30** / 반합 **+15** → 연상 경향
- 배우자궁-시지 육합 **+30** / 반합 **+15** → 연하 / 케어받는 포지션 경향
- 차이 미미하면 노출 안 함.

### 7.4 배우자궁 복수 해석 (daniel 2026-07-21 결정 — 미정 해소)

- **재혼(배우자궁 복수)** → **유료 애정 리딩(`buildLovePrompt`)에 위임**. 무료 카드(`SpouseDualCards`)는 미표기(민감 도메인 §4 회피·"가능성으로만" stance와 충돌 방지).
  - 처리 방식: 이 구조가 *뚜렷할 때만* 기존 흐름/timing 서술에 **"가능성" 톤**으로 녹인다(새 출력 스키마 키 없음 — 별도 항목 만들지 않음).
  - 신호(daniel 기존 stance 재사용, 발명 아님): ① 배우자궁(일지) 충·형·파·해 **반복 개폐**(`age_gap_rule`: "배우자궁/배우자성 충 → 재혼") ② 배우자성 정·편 **혼잡** ③ (여명) **상관견관**(관=배우자 손상).

## 8. 이어짐 관법 — 짝사랑/대시 '성사·발전' 판정 (daniel 2026-07-22 확정·ground truth)

두 관계 유형이 실제로 **이어질/성사될지** 판정. timing(재회/애정)이 아니라 *이 관계가 발전하느냐*의 구조 신호.

### 8.1 공통 필요조건 (★최종 게이트)
- **배우자궁(일지)이 세운과 합으로 열림**(육합/삼합=반합) = 실제 정착의 필요조건.
- 배우자궁이 **충·파·원진으로 흔들리면 → 미성사**(발생 경로 무관).
- 주도권(식상 vs 배우자성)은 "발생 경로"만 다르고, **정착 여부는 항상 일지(배우자궁) 상태가 결정**.

### 8.2 내가 좋아함 (짝사랑 → 성사) — ★남/여 비대칭
주도권=나. **식상(食傷)** = 내가 표현·다가가는 힘. 원국에 식상 없으면 이 유형은 타고나길 약함 → 세운 식상 발동 시기에만 일시적.
- **남명** (배우자성=재성·일간이 *극하는* 대상): 식상生財로 순조롭게 "다가가 잡는" 힘.
  - **이어짐 = 세운 식상 발동 + 배우자궁 합 + 배우자성(재)이 그 해 충/극당하지 않음(가드)**.
  - ※식상生財는 오행상 방향(GEN[식상]=재)은 *항상* 성립하나, 재성이 동시에 충맞으면 생하는 기운이 갈 곳 잃고 흩어짐 → **"재성 파괴 안 됨"이 조건**(재성이 적극 활성일 필요는 없음·식상이 밀어줌).
- **여명** (배우자성=관성·나를 *극하는* 대상): 식상이 관을 만나면 **상관견관**(내 표현이 상대 권위/관계 자체를 공격).
  - → 식상을 배우자성에 *직접 연결 금지*. **재성을 통관(通關)**으로 끼움: 식상生財·財生官 2단계로 극 우회(자평진전 상관견관 완화법).
  - **이어짐 = 세운 식상 발동 + 재성 통관 존재(생조받거나 최소 파괴 안 됨) + 관성이 그 재를 받아 배우자궁과 합**.
  - ※재성 통관 없이 식상만 강하게 뜨면 = 상관견관 = "연애는 생겨도 **정착과 반대**(불안정·안정 관계 밀어냄)".

### 8.3 상대가 좋아함 (대시 받음 → 성사)
주도권=상대. 도화·배우자성이 나를 향해 옴(내가 뭘 안 해도).
- **이어짐 = 배우자성이 세운 합으로 활성화 + 배우자궁(일지) 합 열림**.
- 배우자성만 활성 + 배우자궁 닫힘/흔들림(파·원진·충) = **"인기는 있는데 진지한 관계로 안 이어짐"**.

### 8.4 검증 (daniel 辛丑 남명)
- **2032 壬子**: 세운 식상(壬子=水) 발동 + 子丑육합(배우자궁 열림) → **내가좋아함 이어짐 ✓**(식상 최초 등장).
- **2027 未**: 卯未반합(배우자성 활성) + 丑未충(배우자궁 흔들림) → **상대가좋아함 미성사="인기만" ✓**.

### 8.5 부분 상태(미성사 결)
- 내가좋아함: 식상 있고 궁 안 열림 → "마음만/표현 못함" · 표현했으나 궁 흔들림 → "표현했지만 안착 안 됨".
- 상대가좋아함: 배우자성 활성 + 궁 흔들림 → "인기만·정착 못 감".
