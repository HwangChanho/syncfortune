# SyncFortune プライバシーポリシー

**最終更新日: 2026年6月17日**

> ⚠️ ドラフトです。公開前に法務確認が必要です。`[ ]` は運営者が記入する項目です。（韓国「個人情報保護法」および Apple App Store のプライバシー要件に基づく）

SyncFortune（以下「運営者」）は、韓国「個人情報保護法」等の関連法令を遵守し、利用者の個人情報を以下のとおり取り扱います。

## 1. 収集する個人情報の項目および目的
| 項目 | 目的 | 備考 |
|---|---|---|
| 生年月日・出生時刻・性別・出生地 | 四柱推命・紫微斗数の命式計算および鑑定生成 | **サーバー保存時に暗号化**、復号は運営者権限に限定 |
| メール・ソーシャルログイン識別子 | アカウント作成・認証、有料利用券の帰属 | |
| 決済・購入履歴 | アプリ内課金の処理 | カード情報等は運営者が保有しない |
| 広告識別子（IDFA等） | 広告の配信 | Google AdMob、可能な場合は非パーソナライズ広告 |
| 端末情報・利用ログ | エラー診断・サービス改善 | 収集日から30日後に自動削除 |

※ 運営者は利用者が入力または同意した情報のみを収集します。本人以外（家族・知人等）の命式を登録する場合、利用者は当該者の同意を得る責任があります。

## 2. 個人情報の利用目的
- 占いコンテンツ（AI鑑定）の生成・提供
- アカウント管理、本人認証、有料利用券の管理
- 決済処理および広告配信
- サービス品質改善、エラー対応、不正利用防止

## 3. 委託および国外移転
サービス提供のため、以下に個人情報の処理を委託し、一部の情報は国外サーバーに保存・処理される場合があります。

| 受託者 | 委託業務 | 取扱項目 |
|---|---|---|
| Supabase Inc. | データベース・認証 | 命式（暗号化）・メール・利用ログ |
| Anthropic, PBC（Claude API） | 鑑定の生成 | 命式データ（計算用） |
| RevenueCat, Inc. / Apple Inc. | アプリ内課金・検証 | 決済・購入履歴 |
| Google LLC（AdMob） | 広告配信 | 広告識別子 |

## 4. 保有期間および破棄
- **アカウント・命式データ**: アカウント削除時に遅滞なく破棄します。
- **利用ログ**: 収集日から30日経過後に自動破棄します。
- 法令により保存が必要な場合は、当該期間保管後に破棄します。

## 5. 利用者の権利
- 利用者はアプリ内の **設定 > アカウント削除** から、本人の個人情報を直接削除できます。
- 個人情報の閲覧・訂正・処理停止の請求は下記の連絡先から行え、運営者は法令に従い遅滞なく対応します。

## 6. 安全性確保のための措置
- 生年月日等のセンシティブな情報はサーバー上で暗号化して保存します。
- 行レベルのアクセス制御（RLS）と通信区間の暗号化（HTTPS）を適用します。
- 個人情報へのアクセス権限を最小限の人員に制限します。

## 7. 満14歳未満の児童
本サービスは満14歳未満の児童を主な対象としていません。満14歳未満の登録には法定代理人の同意が必要です。
